# AI Content Factory Platform

## Proyecto piloto

México Hoy en 5 Minutos

## Objetivo

Crear una plataforma de producción de contenido con Inteligencia
Artificial que permita investigar, generar, producir, publicar y
analizar contenido digital de manera escalable.

## Versión

v0.1 MVP - Validación inicial del flujo completo.

## Componentes principales

-   Motor de investigación
-   Generador de guiones IA
-   Producción multimedia
-   Publicación YouTube
-   Analítica y mejora continua
